# ReadingRecord
📚 책 읽고 정리하기 📚

**모르는 내용을 공유하고, 내가 아는 내용도 공유하자**  
**한 문장이라도 모르는 것을 대충 넘어가지 말자**

## 규칙

* 주 단위로 책을 정리한다.
* 분량은 주당 책의 4장~5장 정도로 유동적으로 정한다.

## 스터디 방향성

1. 각자 스터디 시간에 모여서 책을 읽고 정기적으로(일요일 저녁 20시) 회고 및 피드백하는 시간을 갖는 방식
2. 한 줄 설명 서로 물어보기
3. 각 챕터 정리 후 토요일까지 업로드

**정기적 모임을 통해서 차주 스터디 방향성을 보완**
